# Weather Data Visualizer

## Description
This is a python project to analyze weather data. I used pandas to clean data and Matplotlib to make charts.

## Tools that i used 
- python
- Pandas
- NumPy
- Matplotlib

## How to Run
1. Install these libraries: `pip install pandas matplotlib numpy`
2. Run the script: `python weather_analysis.py`
3. Check the `plots/` where png output generated.
