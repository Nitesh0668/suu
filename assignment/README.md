# Smart Event Dashboard

A modern interactive event-management dashboard built using **HTML, CSS, and JavaScript**.
This project demonstrates **DOM manipulation, event handling, UI animations, and dashboard layout design**.

---

## Preview

Interactive dashboard with:

* Event creation form
* Dynamic event cards
* Sample events generator
* DOM key-press demo
* Animated UI with gradient background

---

## Features

* Add new event
* Delete event card
* Clear all events
* Add sample events
* Keyboard event display
* Hover animations
* Gradient button UI
* Background image dashboard layout

---

## Tech Stack

| Technology | Purpose              |
